# Как Открыть И Протестировать

## Готовые варианты

- Готовая тестовая база с уже загруженной конфигурацией:
  `C:\Users\Admin\Documents\InfoBase\1c_v5\_verify_ui_dump_db`
- Загружаемый дамп конфигурации:
  `C:\Users\Admin\Documents\InfoBase\1c_v5\_real_loadable_dump`
- Контрольная обратная выгрузка из успешно загруженной базы:
  `C:\Users\Admin\Documents\InfoBase\1c_v5\_battle_ui_after`

Не загружайте вручную папку:
`C:\Users\Admin\Documents\InfoBase\1c_v5\_battle_rights`

Это служебные файлы прав, не конфигурация. В ней нет `Configuration.xml`.

## Самый быстрый старт

1. В окне запуска 1С нажать `Добавить`.
2. Выбрать `Добавление в список существующей информационной базы`.
3. Выбрать `Файловая информационная база`.
4. Указать каталог:
   `C:\Users\Admin\Documents\InfoBase\1c_v5\_verify_ui_dump_db`
5. Открыть базу сначала в `Конфигураторе`, затем в режиме `1С:Предприятие`.

## Если хотите загрузить конфигурацию в новую базу сами

1. Создать новую пустую файловую ИБ.
2. Открыть ее в `Конфигураторе`.
3. Выполнить `Конфигурация -> Загрузить файлы конфигурации`.
4. Выбрать каталог:
   `C:\Users\Admin\Documents\InfoBase\1c_v5\_real_loadable_dump`
5. После загрузки выполнить `Конфигурация -> Обновить конфигурацию базы данных`.
6. Запустить базу в режиме `1С:Предприятие`.

## Что проверять в интерфейсе

- `Заказы -> Заказ`: список заказов, создание нового заказа, табличная часть `Позиции`
- `Заказы -> ЗаказПоставщику`: список заказов поставщику, табличная часть `Позиции`
- `Логистика -> Отправка`: список отправок, табличные части `Позиции` и `Грузоместа`
- `НСИ -> Товары`: список товаров, элемент товара, табличная часть `КроссНомера`
- `НСИ -> МаршрутыДоставки`: элемент маршрута, табличная часть `Точки`
- `Финансы -> ОплатаЗаказа`: ввод оплат клиента, поставщика и перевозчика
- `Отчеты`: команды отчетов `РеестрЗаказов`, `КонтрольОплат`, `ЛогистическийОтчет`, `МаржинальностьЗаказов`, `ДвижениеДенежныхСредствПоЗаказам`, `НезавершенныеЗаказы`

Пока используются стандартные управляемые формы 1С. Они должны открываться из разделов и показывать все реквизиты, табличные части и стандартные команды создания/записи.

## Подтверждение загрузки

Успешная загрузка подтверждена обратной выгрузкой из тестовой ИБ:

- раздел `Заказы` содержит `Document.Заказ`, `Document.ЗаказПоставщику`, `Catalog.Заказчики`, `Catalog.Товары` и отчеты по заказам
- раздел `Логистика` содержит `Document.Отправка`, справочники логистики и логистический отчет
- раздел `Финансы` содержит `Document.ОплатаЗаказа`, `Document.СопутствующийРасход` и финансовые отчеты
- в `Заказ.xml` есть табличная часть `Позиции`
- в `Отправка.xml` есть табличные части `Позиции` и `Грузоместа`
- в `Товары.xml` есть табличная часть `КроссНомера`
- в `МаршрутыДоставки.xml` есть табличная часть `Точки`
